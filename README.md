# GPT-Image-2 Web

一个纯浏览器端的 AI 图片生成工具，通过 OpenAI 标准接口调用 GPT-Image-2 模型。

**零后端、零部署、零数据上传** — 打开网页就能用。

---

## 核心特性

### 兼容 OpenAI 标准接口

通过 `/v1/images/generations` 和 `/v1/images/edits` 标准接口调用，兼容任何实现了 OpenAI Images API 的服务商。接口路径可自定义，适配各类代理和中转服务。

### 文生图 + 图生图

- **文生图**：输入提示词，选择尺寸和质量，一键生成
- **图生图**：上传参考图 + 提示词，对图片进行编辑和修改
- 支持拖拽上传参考图，生成结果也可直接拖拽为下一次的参考图

### 内置 AI 抠图

浏览器端本地运行的 AI 抠图，基于 `@imgly/background-removal` 模型。支持 GPU / CPU 模式和 Small / Medium / Large 三档精度。**模型在浏览器内加载运行，图片不会上传到任何服务器。**

### 并发队列生成

生成多张图片时自动拆分为独立请求并发执行，每完成一张实时展示结果。支持同时提交多个任务，主画布实时显示进度、计时和已完成的缩略图。可随时取消进行中的任务。

### 纯本地数据存储

所有数据存储在浏览器本地：
- **图片**：IndexedDB 永久存储，不依赖远程服务器
- **配置**：API 地址和密钥保存在 localStorage
- **历史**：提示词历史、生成记录完整保留

只要不清除浏览器数据，你的图库和配置会一直在。

### 提示词管理

独立的「提示词」页面，自动按提示词文本分组展示所有历史产出。相同提示词多次生成的图片合并到一起，一目了然地对比不同参数下的效果差异。

### 多配置管理

支持创建多个 API 配置（不同服务商、不同令牌），一键切换。每个配置可独立设置 API 地址、密钥和自定义接口路径。

## 技术栈

React 19 · MUI 9 · TypeScript · Vite · IndexedDB · @imgly/background-removal

---

## 快速开始

```bash
pnpm install
pnpm dev
```

打开浏览器访问 `http://localhost:5173`，在设置中填入你的 API 地址和密钥即可使用。

## 更新日志

### v0.2.0 (2026-05-18)

#### 新功能
- **提示词页面**：新增「提示词」选项卡，按提示词分组展示所有历史产物，相同提示词的图片自动合并，支持一键复制提示词
- **自定义 API 路径**：配置中新增文生图路径（`/v1/images/generations`）和图生图路径（`/v1/images/edits`）的自定义设置，兼容不同 API 代理
- **多张生成并发**：生成多张图片时并发发送独立请求（每次 `n=1`），支持渐进式展示已完成的图片
- **画布实时进度**：运行中的任务直接在主画布顶部展示进度条、计时和已完成缩略图

#### 修复
- **双存 bug**：修复手动关闭队列任务时图片重复保存到图库的问题
- **图片下载失败**：改用 `response_format: b64_json` 避免跨域资源服务器的防盗链拦截

#### 优化
- **黑曜石主题**：深色模式更换为黑曜石风格（深黑底色 + 暗色渐变高光），浅色模式同步调整为低饱和暖灰
- **Lightbox 按钮**：预览弹窗的关闭和导航按钮改为深色实心背景 + 白色图标，浅色/深色模式下均清晰可见
- **图片网格**：结果展示改为响应式网格布局，hover 微抬起 + 阴影反馈
