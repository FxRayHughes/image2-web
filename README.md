# GPT-Image-2 生图工具

基于 React + MUI 的 GPT-Image-2 图片生成前端工具。

## 更新日志

### v0.2.0 (2026-05-18)

#### 新功能
- **提示词页面**：新增「提示词」选项卡，按提示词分组展示所有历史产物，相同提示词的图片自动合并，支持一键复制提示词
- **自定义 API 路径**：配置中新增文生图路径（`/v1/images/generations`）和图生图路径（`/v1/images/edits`）的自定义设置，兼容不同 API 代理
- **多张生成并发**：生成多张图片时并发发送独立请求（每次 `n=1`），支持渐进式展示已完成的图片
- **画布实时进度**：运行中的任务直接在主画布顶部展示进度条、计时和已完成缩略图

#### 修复
- **双存 bug**：修复手动关闭队列任务时图片重复保存到图库的问题
- **图片下载失败**：改用 `response_format: b64_json` 避免跨域资源服务器的防盗链拦截

#### 优化
- **黑曜石主题**：深色模式更换为黑曜石风格（深黑底色 + 暗色渐变高光），浅色模式同步调整为低饱和暖灰
- **Lightbox 按钮**：预览弹窗的关闭和导航按钮改为深色实心背景 + 白色图标，浅色/深色模式下均清晰可见
- **图片网格**：结果展示改为响应式网格布局，hover 微抬起 + 阴影反馈

### v0.1.0 (2026-05-17)

- 初始版本：文生图 / 图生图 / AI 抠图 / 图库 / 多配置管理
